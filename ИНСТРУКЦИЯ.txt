ДЛЯ АДМИНИСТРАТОРА (запуск сервера):
─────────────────────────────────────
1. Поместите все файлы в одну папку:
   - PMO_Control_Center.html
   - server.js
   - ЗАПУСТИТЬ_СЕРВЕР.bat

2. Дважды кликните на ЗАПУСТИТЬ_СЕРВЕР.bat

3. В окне появится ваш IP-адрес, например:
   http://192.168.1.50:8080

4. Отправьте эту ссылку всем сотрудникам.

5. НЕ ЗАКРЫВАЙТЕ окно — пока оно открыто,
   сервер работает.

═══════════════════════════════════════════════════

ДЛЯ СОТРУДНИКОВ (вход в систему):
───────────────────────────────────
1. Откройте браузер (Chrome, Edge, Firefox)

2. Введите в адресную строку ссылку от админа:
   http://192.168.X.X:8080
   (точный адрес спросите у администратора)

3. На экране входа введите:
   - Ваше имя (например: Алия Ю.)
   - Пароль (выдаёт администратор)

ЕСЛИ НЕ ОТКРЫВАЕТСЯ (настройка файрвола):
──────────────────────────────────────────
1. Пуск → Поиск → "Брандмауэр Windows"
2. "Дополнительные параметры"
3. "Правила для входящих подключений" → "Создать"
4. Тип: Порт → TCP → Порт: 8080
5. Разрешить подключение → Готово
