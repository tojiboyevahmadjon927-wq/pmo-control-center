ШАГ 1 — ПОЛУЧИ КЛЮЧ CLAUDE API
────────────────────────────────
1. Открой https://console.anthropic.com
2. Войди или зарегистрируйся
3. API Keys → Create Key
4. Скопируй ключ (начинается с sk-ant-...)
5. Вставь в agent_config.json → "claude_api_key"

ШАГ 2 — СОЗДАЙ TELEGRAM БОТА
──────────────────────────────
1. Открой Telegram, найди @BotFather
2. Напиши: /newbot
3. Придумай имя: PMO Turon Bot
4. Придумай username: pmo_turon_bot
5. BotFather даст токен — вставь в "telegram_token"

ШАГ 3 — УЗНАЙ СВОЙ TELEGRAM ID
────────────────────────────────
1. Напиши боту @userinfobot
2. Он ответит твоим chat_id
3. Вставь в "admin_telegram_ids": ["ТВОй_ID"]

ШАГ 4 — ДОБАВЬ IDs КОМАНДЫ
────────────────────────────
Попроси каждого менеджера:
1. Написать @userinfobot
2. Сообщить тебе их chat_id
Вставь в "telegram_chat_ids"

ШАГ 5 — ЗАПУСК
────────────────
Запусти ЗАПУСТИТЬ_АГЕНТА.bat
(можно одновременно с основным сервером)
